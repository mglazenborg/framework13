# Elphael

Installation script for Arch Linux on my Framework 13 Pro (Elphael).

Mostly based on Omarchy, with some changes and less bloat out of the box.

# Start installation

Run the command below to start the installation.
I know it's a long url :/

```bash
eval "$(curl -fsSL https://raw.githubusercontent.com/mglazenborg/elphael/refs/heads/main/boot.sh)"
```
